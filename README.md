# 📈 Análise de Séries Temporais da Produção Industrial com Power BI e IA

Projeto desenvolvido durante o curso da Data Science Academy com o objetivo de aplicar técnicas de Business Intelligence, Inteligência Artificial e Análise de Séries Temporais para monitorar e prever a produção industrial ao longo do tempo.

## 🎯 Objetivo

Construir um dashboard analítico capaz de:

- Monitorar a evolução da produção industrial.
- Comparar desempenho por faixa etária dos colaboradores.
- Identificar padrões históricos.
- Detectar anomalias automaticamente.
- Realizar previsões futuras utilizando recursos nativos de IA do Power BI.

---

## 🛠 Tecnologias Utilizadas

- Power BI
- Power Query
- DAX
- Inteligência Artificial do Power BI
- Análise de Séries Temporais
- Forecasting
- Excel
